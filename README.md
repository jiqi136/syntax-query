# syntax-query
Lightweight syntax query tool for common programming languages. Fast syntax rule lookup, practical interface, fuzzy matching, and quick copy. Helps developers quickly recall or learn syntax details, boosting coding efficiency.轻量级编程语言语法查询工具。支持常见语言的语法规则快速检索。可通过 实用便捷 界面交互，内置模糊匹配与快捷复制功能，帮助开发者快速回忆或学习语法细节，提升编码效率。
